このフォルダの demo-tone-01.mp3 と demo-tone-02.mp3 は、
音声プレイヤーの動作確認用に作られた短いサンプル音です。

公開前に、ご自身のMP3へ差し替えてください。
ファイル名には半角英数字とハイフンを使うことを推奨します。

例：
01-first-light.mp3
02-night-rain.mp3

差し替え後は index.html の次の2か所を新しいファイル名に変更します。

<source src="audio/新しいファイル名.mp3" type="audio/mpeg">
<a href="audio/新しいファイル名.mp3" download>
